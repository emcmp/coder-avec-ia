# Gabarits pédagogiques

> Source : `project/conventions/gabarits-pedagogiques.md`  
> Identifiant : `[DOC-CHR-005]`  
> Version : 0.4  
> Dernière modification : 2026-06-12  
> Statut : convention de travail initiale

Cette page présente les gabarits réutilisables pour produire les contenus pédagogiques du parcours **Coder avec l’IA**.

## Règle principale

Une page publique doit rester lisible pour l’étudiant.

Les identifiants, statuts et compétences restent disponibles, mais ils ne doivent pas dominer la page.

## Découplage

Le contenu, le parcours et l’outil d’exemple sont trois choses différentes.

- Le contenu garde un titre stable.
- L’ordre de rencontre appartient à la navigation ou à l’index.
- Le provider utilisé dans les exemples n’est pas automatiquement le sujet de la notion.

## Standard des rencontres publiques

Structure recommandée :

- titre stable;
- métadonnées discrètes;
- courte amorce;
- sections avec emoji;
- exemple de prompt ou réponse IA;
- explication guidée;
- à essayer;
- récapitulatif;
- pour aller plus loin;
- références pédagogiques repliées.

## Boîtes IA

Tous les prompts, réponses simulées, extraits de sortie IA et résumés de modifications doivent utiliser une boîte IA.

Classes principales :

```text
aiBox
aiBoxPrompt
aiBoxResponse
aiBoxChanges
```
