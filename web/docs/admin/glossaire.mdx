# Glossaire

> Source : `project/glossaire/README.md`

Cette page servira à réviser les entrées de glossaire du projet.

Le glossaire utilisera des identifiants stables de type `GLO`, par exemple `GLO-IA-001`, `GLO-CON-001` ou `GLO-MAP-001`.

Les entrées détaillées seront ajoutées progressivement lorsque la table des matières et le catalogue des blocs seront stabilisés.
