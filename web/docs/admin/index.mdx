# Admin

Cette section rend visibles dans le site les documents de travail du dossier `project/`.

Elle sert à réviser la structure du projet, les conventions, le catalogue et les décisions avant de transformer le contenu en pages publiques de la trousse.

## Règle de travail

Les fichiers sources dans `project/` demeurent la référence de travail.

Les pages `Admin` servent à rendre ces documents plus faciles à consulter et à réviser dans le site.
