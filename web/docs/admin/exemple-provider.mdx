# Exemple provider

Cette page montre comment présenter une même action dans plusieurs outils IA.

## Principe

L’ordre recommandé est toujours :

```text
GitHub Copilot
Codex
Claude Code
```

GitHub Copilot est le provider de référence.

## Exemple : voir le statut

| Provider | À montrer |
|---|---|
| GitHub Copilot | L’endroit principal où voir le statut ou l’état de l’assistant. |
| Codex | L’équivalent pour suivre une tâche, un usage ou un contexte. |
| Claude Code | L’équivalent pour une session CLI ou locale. |

## Règle

Le panneau provider doit répondre à une question pratique.

Il ne doit pas devenir un comparatif général de produits.
