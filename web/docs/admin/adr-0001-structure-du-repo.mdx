# ADR-0001 — Structure du dépôt

> Source : `project/adr/ADR-0001-structure-du-repo.md`  
> Statut : acceptée

## Décision

Le dépôt distingue deux espaces :

- `project/` pour les documents de conception;
- `web/` pour le site Docusaurus.

## Raison

Les plans, décisions, specs, prompts et notes doivent pouvoir évoluer avant d’être publiés dans le site.

## Conséquence

Le contenu est conçu dans `project/`, puis publié dans `web/` seulement lorsqu’il est prêt.
