# Catalogue des blocs pédagogiques

> Source : `project/catalogue/blocs-pedagogiques.md`  
> Version : 0.1  
> Statut : index de travail

Ce document sert d’index central pour les blocs pédagogiques du projet.

## Rôle du catalogue

Le catalogue permettra de retrouver rapidement :

- les notions;
- les blocs théoriques;
- les exercices;
- les démonstrations;
- les aides visuelles;
- les vidéos;
- les prompts;
- les entrées de glossaire.

## Format souhaité

Chaque entrée devra contenir au minimum :

- un identifiant stable;
- un type;
- un domaine;
- un titre;
- un statut;
- une source ou un emplacement prévu.
