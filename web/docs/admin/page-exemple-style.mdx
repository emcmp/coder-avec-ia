# Page exemple de style

Cette page sert de référence visuelle pour les futures pages du site.

Elle ne présente pas encore un vrai contenu de cours. Elle montre plutôt comment structurer une page, écrire les titres, utiliser les encadrés et présenter du code.

## 1. Intention de la page

Commencer par une courte intention aide le lecteur à comprendre pourquoi la page existe.

:::info
Cette page est un gabarit visuel. Elle sert à valider la forme avant de développer le contenu pédagogique.
:::

## 2. Section courte avec explication

Une section devrait rester courte et traiter une seule idée principale.

Lorsque plusieurs idées apparaissent dans une même section, il vaut mieux créer des sous-sections.

### Exemple de sous-section

Une sous-section sert à préciser un point sans créer une nouvelle page.

:::tip
Une bonne page est facile à balayer rapidement avant d’être lue en détail.
:::

## 3. Numérotation

Utiliser une liste numérotée lorsque l’ordre compte :

1. observer le problème;
2. formuler une hypothèse;
3. tester une solution;
4. valider le résultat.

Utiliser une liste à puces lorsque l’ordre ne compte pas :

- clarté;
- cohérence;
- lisibilité;
- réutilisabilité.

## 4. Encadrés

:::info
Utiliser `info` pour une précision utile.
:::

:::tip
Utiliser `tip` pour un conseil concret.
:::

:::warning
Utiliser `warning` pour une confusion fréquente ou une limite importante.
:::

:::danger
Utiliser `danger` pour une erreur à éviter absolument.
:::

## 5. Bloc de code

Toujours présenter le contexte avant un bloc de code.

Exemple de commande utilisée pour valider le site :

```bash
cd web
npm run build
```

Exemple de structure de fichier :

```text
web/docs/admin/
  charte-visuelle.mdx
  page-exemple-style.mdx
```

## 6. Tableau court

| Élément | Usage recommandé |

## 7. Point de vigilance

:::warning
Ne pas utiliser cette page comme modèle pédagogique final. Elle sert seulement à figer le style visuel.
:::

## 8. Prochaine convention à définir

La prochaine étape sera de définir le style pédagogique : théorie, démonstration, exercice, laboratoire, quiz, aide visuelle et évaluation.
