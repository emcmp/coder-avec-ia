## Objectif

Chaque document important doit pouvoir être classé, identifié, retrouvé et maintenu facilement.

## Champs obligatoires

Chaque document important doit avoir :

| Champ | Rôle |
|---|---|
| ID | Identifiant stable du document |
| Version | Niveau d’évolution du document |
| Dernière modification | Date du dernier changement significatif |
| Statut | Brouillon, actif, stable, en révision, archivé, etc. |
| Source | Fichier source de référence |
| Page Admin | Page publiée, lorsque disponible |
