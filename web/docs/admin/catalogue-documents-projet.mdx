# Catalogue des documents de projet

Cette page présente le registre des documents importants du projet.

La version complète est conservée dans le fichier source suivant :

`project/catalogue/documents-projet.md`

## Objectif

Chaque document important doit pouvoir être classé, identifié et retrouvé facilement.

Les documents de projet incluent notamment :

- les chartes;
- les décisions;
- les catalogues;
- le glossaire;
- le parcours;
- les spécifications;
- le backlog;
- les prompts;
- les archives.

## Règle

Tout nouveau document important doit être ajouté au catalogue source.
