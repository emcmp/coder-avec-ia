# Générer et valider des tests

> Brouillon. Cette activité mettra l'accent sur la génération de tests et leur validation manuelle.

## À couvrir

- Demander des tests pertinents.
- Vérifier ce qu'ils couvrent réellement.
- Corriger les tests insuffisants ou erronés.
