# Bon prompt, mauvais prompt

> Brouillon. Cette activité comparera des formulations de requêtes et leurs effets sur la réponse.

## À couvrir

- Reformuler un prompt imprécis.
- Identifier ce qui manque dans une requête.
- Décrire un prompt de qualité.
