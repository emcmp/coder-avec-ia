# Déboguer avec l'IA

> Brouillon. Cette activité guidera un débogage où l'IA suggère des pistes sans remplacer l'analyse.

## À couvrir

- Lister les symptômes.
- Tester des hypothèses.
- Valider la cause réelle avant de corriger.
