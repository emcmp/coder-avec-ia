# Cours — Coder avec l’IA

Cette section contient les premiers modules de cours publiés.

L’objectif n’est pas de présenter l’IA comme un tour de magie. L’objectif est d’apprendre à travailler avec l’IA dans un vrai contexte de programmation : préciser la tâche, fournir le bon contexte, comprendre où intervenir, puis valider ce qui est produit.

## Progression recommandée

1. **Contexte utile** — apprendre à ne pas faire deviner l’IA.
2. **Cartographie du codebase** — comprendre où agir avant de modifier.
3. **Validation d’une réponse IA** — garder le contrôle sur le résultat.

## Format des modules

Chaque module suit le même format :

- intention;
- compétences visées;
- friction de départ;
- théorie ciblée;
- démonstration courte;
- mini-exercice;
- retour rapide;
- paliers I, II et III.

:::tip
Les modules sont volontairement courts. Ils sont conçus pour être utilisés en classe, en atelier ou comme matériel de préparation.
:::
