## Comment utiliser ces contenus

Les pages publiées ici sont des **notes de recherche structurées** issues des parcours Microsoft Learn. Elles permettent de consulter rapidement la matière, de comparer les notions et d’identifier ce qui mérite d’être repris dans le cours collégial.

Elles ne sont pas encore toutes adaptées au format pédagogique final de **Coder avec l’IA**. Les modules publics du cours continueront d’être retravaillés avec la structure habituelle : mauvaise manière, théorie ciblée, mise en place, expérimentation et récapitulatif.

:::warning Statut des documents
Ces contenus sont des sources de travail. Pour les informations officielles, les mises à jour et les évaluations Microsoft, il faut toujours se référer aux parcours Microsoft Learn d’origine.
:::
