# Fiche « Avant de remettre du code assisté »

> Brouillon. Cette fiche résumera les vérifications à faire avant la remise d'un code soutenu par l'IA.

## À couvrir

- Le code compile ou s'exécute.
- Les tests pertinents passent.
- La déclaration d'usage est complète.
