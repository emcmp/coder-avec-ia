# Vérifier, tester, expliquer

> Brouillon. Cette page montrera la séquence à suivre après un résultat généré ou suggéré par l'IA.

## À couvrir

- Vérifier la logique et les cas limites.
- Tester le code dans un environnement réel.
- Expliquer le résultat avec ses propres mots.
