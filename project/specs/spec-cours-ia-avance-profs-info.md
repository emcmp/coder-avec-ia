# Spec — Formation avancée IA pour profs d’informatique

> État : brouillon minimal.

## Objectif

Définir une formation technique avancée sur l’utilisation de l’IA pour des profs d’informatique.

## Public cible

- Enseignants en informatique.
- Profils techniques intéressés par les outils IA appliqués au développement, à la préparation de cours et à l’évaluation.

## Sujets pressentis

- Context engineering.
- Skills et instructions réutilisables.
- Spec Kit.
- RAG.
- CodeGraph.
- CLI IA.
- Agents dans les dépôts GitHub.
- MCP.
- Évaluation.
- Observabilité.
- Sécurité.
- Modèles locaux.

## À préciser

- Durée de la formation.
- Format des rencontres.
- Démonstrations à retenir.
- Activités pratiques.
- Outils officiellement recommandés.
