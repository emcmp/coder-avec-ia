# Inbox de recherche — Cours IA

Ce dossier reçoit la matière brute issue des recherches approfondies avant tri, fusion et intégration dans le contenu officiel du cours.

## Workflow

```text
Recherche approfondie
        ↓
Synthèse Markdown
        ↓
project/inbox/
        ↓
Tri, validation et fusion
        ↓
Contenu de cours
```

## Structure attendue d’un document de recherche

Chaque document devrait contenir, selon le sujet :

- résumé exécutif;
- concepts clés;
- idées fortes;
- meilleures pratiques;
- outils et commandes;
- exemples concrets;
- laboratoires possibles;
- pièges fréquents;
- sujets avancés;
- ressources officielles;
- idées à intégrer au cours;
- informations encore manquantes.
