# OpenAI Cookbook

Collection of example notebooks for working with OpenAI models and APIs.

## Topics

- **Agents**: Multi-agent workflows, tool calling, orchestration
- **Evals**: Building resilient prompts with evals, grading
- **Multimodal**: Vision, image generation, audio
- **Text**: Prompts, few-shot learning, structured outputs
- **Guardrails**: Safety, moderation, input validation
- **Optimization**: Cost reduction, latency optimization
- **Codex**: Code generation and review
- **GPT-OSS**: Open-source model fine-tuning
