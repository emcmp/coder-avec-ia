# Agent Workflow Evaluation

## Trace grading

1. Open **Logs** > **Traces** in the dashboard
2. Inspect a representative workflow trace
3. Create a grader and run it against the selected traces
4. Use results to refine prompts, tools, routing, or guardrails

## Datasets and eval runs

For repeatable evaluations:
- Create datasets from traces
- Run eval runs to benchmark changes
- Compare prompts and configurations
