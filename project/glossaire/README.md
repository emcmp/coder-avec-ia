# Glossaire

> Version : 0.1  
> Statut : espace de travail

Ce dossier servira à construire le glossaire du projet **Coder avec l’IA**.

Les entrées de glossaire doivent utiliser le type `GLO` défini dans la nomenclature.

## Format recommandé

```markdown
## [GLO-DOMAINE-NNN] Terme

**Définition courte**  
Définition en une ou deux phrases.

**Contexte pédagogique**  
Pourquoi ce terme est important dans le cours.

**À ne pas confondre avec**  
Termes proches ou erreurs fréquentes.

**Exemples liés**  
Blocs pédagogiques, exercices ou aides visuelles associés.
```
