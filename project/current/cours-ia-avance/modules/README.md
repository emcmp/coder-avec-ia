## Rôle des modules pilotes

Ces modules doivent permettre de vérifier :

- si le format de micro-module est assez léger;
- si les paliers I / II / III sont utiles;
- si la place de GitHub Copilot comme outil principal fonctionne;
- si les équivalences Codex et Claude Code restent courtes;
- si les exercices sont suffisamment concrets;
- si les compétences observables sont bien liées aux modules;
- si les modules peuvent ensuite être publiés comme pages de cours.

## Prochaine étape

Après validation, ces modules pourront servir de modèle pour produire les autres micro-modules du catalogue.
