# Catalogue des blocs pédagogiques

> Version : 0.1  
> Statut : index de travail

Ce fichier servira d’index central pour les blocs pédagogiques du projet.

Les identifiants doivent respecter la convention définie dans :

```text
project/conventions/nomenclature-blocs-pedagogiques.md
```

## Rôle du catalogue

Le catalogue permet de retrouver rapidement :

- les notions;
- les blocs théoriques;
- les exercices;
- les démonstrations;
- les aides visuelles;
- les vidéos;
- les prompts;
- les entrées de glossaire.

## Format recommandé

```markdown
| ID | Type | Domaine | Titre | Statut | Source |
|---|---|---|---|---|---|
| [THE-CON-001] | Théorie | CON | Pourquoi connecter un agent à des outils? | draft | À créer |
```
